🚀 StratumX – Fullstack Corporate & E-Commerce Platform

StratumX is a modern full-stack web platform that combines a corporate website, e-commerce store, and admin dashboard into a single scalable system.

Built with performance, scalability, and modern architecture in mind, StratumX allows businesses to showcase their services, sell products online, and manage everything from one powerful dashboard.

✨ Features
🏢 Corporate Website

Modern landing page showcasing the brand

About page with company vision and technologies

Services page displaying offered technical services

Contact form for direct communication

🛒 E-Commerce Store

Browse products with search and category filters

Product details page with quantity selection

Shopping cart with order summary

Secure online payments via Stripe

⚙️ Admin Dashboard

Product management (Create / Update / Delete)

Order management and status tracking

Manage services and content

Secure authentication using JWT
